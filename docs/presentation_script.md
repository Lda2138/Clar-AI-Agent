# 🎤 《随机信号分析》AI 助教 Clar — 五分钟项目汇报演讲稿

**预计时长**：5 分钟
**演讲受众**：评委老师、同学或技术团队
**配套演示**：建议在演讲过程中，安排一位助手在旁边操作网页（展示生成信号、悬停图谱、小球问答等交互）。

---

## 第一部分：开场与痛点引入 (约 1 分钟)

**【幻灯片 1：项目封面图，展示系统 UI 截图与名称】**

**各位评委、老师、同学们，大家好！**
今天我非常荣幸向大家展示我们的项目——**《随机信号分析》AI 助教 Clar**。

大家如果学过《随机信号分析》这门课，肯定都有过这样的体会：这门课数学抽象度极高、公式推导密集。诸如广义平稳、维纳-欣钦定理、卡尔曼滤波等概念，仅仅看书本上的公式很难建立起物理直觉。而传统教学中，理论与实践往往是脱节的，学生在疑惑时，也无法立刻看到参数调整对信号产生的影响。

为了解决这个痛点，我们开发了这款集**“感知 — 决策 — 控制”**闭环于一体的智能教学系统。它不仅是一个能陪你聊天的 AI，更是一个能帮你“画图”、“调参”、“推公式”的全能仿真实验室。

---

## 第二部分：核心亮点与功能展示 (约 1.5 分钟)

**【幻灯片 2：UI 界面与核心功能，展示信号工坊和双交互模式】**

我们的系统最大的亮点在于**“交互零摩擦”**。

在传统软件里，你需要手动寻找各种按钮去设置频率、信噪比等参数。而在我们的系统中，你只需要对 AI 说一句：“帮我生成一个 200Hz 的含噪正弦波，并做一次维纳滤波”，AI 就能直接理解你的自然语言，自动在后台填装参数、跨页面跳转，并瞬间渲染出时域和频域的 6 种专业图表。我们称之为**“口令即操作”**。

此外，我们在工程交互上做了两个非常亮眼的创新：
1. **双交互模式**：我们不仅有经典的侧边栏对话框，还首创了**“Clar-ball 悬浮精灵模式”**。这个像呼吸灯一样的小球会智能感知你鼠标在屏幕上的停留位置。比如当你鼠标停在“幅度谱”上，或者停在“知识图谱”的某个节点上时，向它提问，它就能精准知道你正在看什么，给出有针对性的解答。
2. **主动式缺陷诊断**：当你输入的采样率不满足奈奎斯特采样定理，或者卡尔曼滤波发散时，小球会敏锐地捕捉到物理异常，闪烁紫光，并在右下角主动弹出一张极具质感的**“诊断卡片”**，告诉你错在哪里，并引导你修复。

## 第三部分：系统架构与硬核技术 (约 1.5 分钟)

**【幻灯片 3：架构图设计，展示前后端分离与大模型接入】**

这么丝滑的体验背后，是我们坚实的底层架构。我们采用了前轻后重、高度解耦的设计：
- **前端**我们完全采用原生 JavaScript 编写，摒弃了厚重的框架，保证了极高的运行效率和易部署性。UI 上我们大面积采用了前沿的 **Glassmorphism（玻璃拟态）** 设计语言，赋予了学术严肃性一种高级的科技美感。
- **后端**采用 FastAPI，它作为 API 网关，串联了强大的信号计算引擎和知识库。我们手写了 9 种信号发生器和 4 种经典滤波算法的纯数学实现。
- **AI 智能体层**：我们接入了最新的大语言模型，并为其配备了 5 个 Function Calling 工具。为了实现极致的响应速度，我们自己实现了一个**流式 JSON 实时解析器**（`JSONReplyStreamParser`），能够在接收到 AI 数据流的瞬间，就将其中的对话内容逐字渲染到屏幕上，把首字延迟降到了毫秒级。

---

## 第四部分：总结与展望 (约 1 分钟)

**【幻灯片 4：总结与致谢】**

总而言之，Clar 助教不仅仅是一个简单的套壳 ChatGPT，它是深度结合了**领域专业知识（Domain Knowledge）**与**大模型工具调用能力（Agent Tools）**的教学平台。

我们通过 10 余项工程设计的巧思，例如边缘羽化蒙版、高分辨率波形双轨重建等，在网页端提供了一个比拟原生 App 般丝滑的信号分析实验室。

我们相信，未来的教育工具就应该是这样：既有严谨的数学内核，又有极具温度和灵性的 AI 交互。希望能通过我们的项目，让更多人轻松跨越《随机信号分析》这座大山！

我的汇报到此结束，感谢大家的聆听！欢迎各位老师批评指正。
