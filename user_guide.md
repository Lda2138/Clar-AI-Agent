## 1. 快速开始

### 1.1 环境准备

**系统要求**：Python 3.10 或更高版本

**安装步骤**：

```bash
# 第一步：安装 Python 依赖
pip install -r requirements.txt

# 第二步：配置 AI 密钥
# 打开项目根目录下的 .env 文件，将你的 DeepSeek API Key 填入：
# DEEPSEEK_API_KEY=sk-你的密钥
```

### 1.2 启动系统

```bash
python main.py
```

看到类似以下输出，说明启动成功：

```
INFO:     Uvicorn running on http://0.0.0.0:8001 (Press CTRL+C to quit)
INFO:     SignalAgent 初始化成功
```

### 1.3 打开页面

在浏览器中访问：**http://localhost:8001**

> **提示**：推荐使用 Chrome 或 Edge 浏览器，以获得最佳的动画和渲染效果。

### 1.4 公网分享（可选）

如果需要让其他人（如评审老师）远程访问，可使用 ngrok：

```bash
ngrok http 8001
```

将生成的 `https://xxxx.ngrok-free.app` 链接发送给对方即可。

## 3. 信号工坊

信号工坊是你的核心实验台，用来生成各种信号并观察它们的统计特性。

### 3.1 生成信号

**方法一：手动调参**

1. 在底栏找到参数控制区域
2. 选择 **信号类型**（下拉菜单）
3. 调整以下参数：
   - **频率** — 信号的基频 / 中心频率（单位：Hz）
   - **采样率** — 模拟采样过程的采样频率（单位：Hz）
   - **信噪比 (SNR)** — 纯净信号与噪声的功率比（单位：dB）
4. 点击 **「生成信号」** 按钮

**方法二：让 AI 帮你生成**（推荐）

直接在右侧聊天框输入自然语言指令，例如：

| 你说的话 | AI 会做什么 |
|---------|-----------|
| "生成一个正弦波" | 自动用经典参数（200Hz, 10000Hz采样, 10dB SNR）生成正弦+白噪声信号 |
| "来一个 500Hz 的方波，信噪比 5dB" | 精确设置 freq=500, snr=5 并生成 |
| "生成窄带信号，带宽 100Hz" | 生成窄带高斯信号，带宽设为 100Hz |
| "生成 LFM 扫频信号" | 生成线性调频信号（100→400Hz） |

### 3.2 支持的 9 种信号

| 信号类型 | 特殊参数 | 教学要点 |
|---------|---------|---------|
| 正弦+白噪声 | — | 最基础的确定性+随机信号模型 |
| 方波+白噪声 | — | 丰富的谐波分量，观察频谱扩展 |
| 三角波+白噪声 | — | 对比方波的谐波衰减特性 |
| 双频正弦+白噪声 | 第二频率 $f_2$ | 观察双频叠加的拍频现象 |
| 窄带高斯信号 | 带宽 $B$ | 平稳窄带随机过程的经典模型 |
| 瑞利分布包络 | 带宽 $B$ | $R = \sqrt{I^2 + Q^2}$ 包络检测 |
| 线性调频(LFM) | 终止频率 $f_{end}$ | 雷达脉冲压缩基础 |
| 高斯白噪声 | — | 纯噪声，观察均匀功率谱 |
| 一阶马尔可夫过程 | 自回归系数 $a$ | 自相关函数指数衰减特性 |

### 3.3 查看分析图表

生成信号后，工作区会显示两组图表：

**时域图表**（左上区域，可切换）：
- **波形图** — 显示纯净信号（蓝色平滑曲线）和含噪信号（红色离散点）
- **自相关函数** — 信号在不同时间延迟下与自身的相关程度
- **互相关函数** — 纯净信号与含噪信号的相关程度
- **概率密度函数 (PDF)** — 信号幅度的统计分布

**频域图表**（左下区域，可切换）：
- **幅度谱** — 纯净信号和含噪信号的频率成分
- **功率谱密度 (PSD)** — 单位频带内的平均功率
- **相位谱** — 各频率分量的相位信息

> **如何切换**：点击图表区域上方的小按钮即可切换不同的图表视图。你也可以告诉 AI："*帮我看一下自相关函数*"，Clar 会自动切换到对应图表。

### 3.4 统计特征指标

生成信号后，右侧会显示实时计算的统计特征：

- **均值** ($E[X]$) — 信号的直流分量
- **方差** ($D[X]$) — 信号的波动程度
- **标准差** ($\sigma$) — 方差的平方根
- **均方根** (RMS) — 信号的有效值
- **峰峰值** — 最大值与最小值之差
- **峭度** — 分布的尖锐程度（高斯分布为 0）

### 3.5 奈奎斯特警告

当你设置的采样率过低时，系统会自动预警：

| 采样率状态 | 输入框颜色 | 含义 |
|-----------|-----------|------|
| $f_s \geq 4 f_{max}$ | 正常 | 采样充分，波形精度高 |
| $2 f_{max} \leq f_s < 4 f_{max}$ | 🟠 橙色 | 接近临界值，重建精度可能不足 |
| $f_s < 2 f_{max}$ | 🔴 红色 | 违反采样定理，将出现频谱混叠！ |

## 4. 知识图谱

知识图谱页面展示了《随机信号分析》课程的完整知识网络。

### 4.1 图谱结构

图谱是一个力导向的树形网络：

```
章节节点（蓝色大圆）
  ├── 小节节点（绿色中圆）
  │     ├── 知识点（黄色/红色小圆）
  │     ├── 知识点
  │     └── 知识点
  └── 小节节点
        └── 知识点
```

- **蓝色大圆**：章节（如"第一章 随机过程的基本理论"）
- **绿色中圆**：小节（如"特殊的随机过程"）
- **黄色/红色小圆**：具体知识点（红色表示重点考核内容）

### 4.2 交互方式

**拖拽和缩放**：
- 鼠标拖拽可以平移图谱视角
- 滚轮缩放可以放大/缩小
- 节点可以自由拖动重新排列

**点击节点**：
1. 点击任意节点
2. AI 会自动为你讲解该节点的内容
3. 右侧会展开 **衍生知识演进面板**，显示：
   - 📌 **基石脉络卡** — 该知识点的上游理论基础
   - 🔥 **当前聚焦新知卡** — 核心概念和工程意义
   - 📐 **公式推导** — 关键公式（LaTeX 渲染）
   - ⚠️ **常见误区** — 考试易错点提醒
   - 💡 **追问按钮** — AI 生成的启发性问题，点击可深入探讨

### 4.3 知识点高亮

当你在聊天中问到某个知识点时，图谱中对应的节点会自动高亮（变大 + 发光），帮助你直观看到当前讨论的上下文位置。

## 5. 工具箱

工具箱是你的信号处理实验室，提供 4 种滤波/估计算法。

> **前提条件**：使用工具箱前，需要先在信号工坊生成一个信号。工具箱会对该信号进行处理。

### 5.1 滑动平均滤波

**作用**：通过对窗口内的样本取平均值来平滑信号，抑制高频噪声。

**操作步骤**：
1. 切换到「工具箱」页面
2. 选择滤波类型为 **滑动平均**
3. 设置窗口大小 $N$（默认 10，越大越平滑但会损失细节）
4. 点击「运行滤波」

**或者告诉 AI**：*"帮我做一个 20 点的滑动平均滤波"*

### 5.2 RC 一阶低通滤波

**作用**：模拟经典的 RC 积分电路，截止频率以下的分量通过，以上的被衰减。

**操作步骤**：
1. 选择滤波类型为 **RC 低通**
2. 设置截止频率 $f_c$（默认 500Hz）
3. 点击「运行滤波」

**或者告诉 AI**：*"截止频率 300Hz，运行低通滤波"*

### 5.3 维纳滤波

**作用**：基于信号和噪声的功率谱密度，在最小均方误差准则下进行最优去噪。

**操作步骤**：
1. 选择滤波类型为 **维纳滤波**
2. 点击「运行滤波」（无需额外参数，自适应计算）

**或者告诉 AI**：*"运行维纳滤波"*

### 5.4 卡尔曼滤波仿真

**作用**：演示经典的目标跟踪场景——在有噪声的测量下，估计运动目标的位置和速度。

**操作步骤**：
1. 切换到卡尔曼仿真面板
2. 调整参数：
   - **过程噪声强度 $q$** — 模型不确定度（越大 → 更信任测量）
   - **测量噪声标准差 $r$** — 传感器精度（越大 → 更信任模型）
   - **初始速度 $v_0$** — 目标的初始运动速度
3. 点击「运行仿真」

**或者告诉 AI**：*"跑一下卡尔曼滤波仿真，q 设为 0.05"*

**查看结果**：
- **位置跟踪对比图** — 真实位置 vs 测量值 vs 卡尔曼估计
- **速度估计对比图** — 真实速度 vs 卡尔曼估计
- **收敛曲线** — 误差协方差 $P_{11}$ 和增益 $K_1, K_2$ 随时间的变化

> **调参建议**：试着改变 $q/r$ 的比值，观察滤波效果的变化——这就是教材里说的"模型与测量的信任博弈"！

## 6. AI 助教对话

### 6.1 基本使用

在右侧的聊天输入框中输入问题，按 **Enter** 发送。

**输入操作**：
| 按键 | 效果 |
|------|------|
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行（不发送） |

### 6.2 你可以问什么

Clar 是一位专业的《随机信号分析》助教，以下是一些提问示例：

**概念理解类**：
- "什么是广义平稳过程？"
- "维纳-欣钦定理的物理含义是什么？"
- "广义平稳和严格平稳有什么区别？"
- "各态历经性是什么意思？"

**公式推导类**：
- "请推导白噪声通过 RC 低通滤波器的输出功率谱"
- "卡尔曼增益公式是怎么来的？"
- "怎么从自相关函数求功率谱密度？"

**信号分析类**：
- "分析一下我刚才生成的信号" ← 会结合实际数据回答
- "这个信号的自相关函数说明了什么？"
- "为什么含噪信号的频谱会展宽？"

**考试避坑类**：
- "期末考试容易在哪些地方出错？"
- "维纳滤波为什么不能直接在实时系统中使用？"
- "卡尔曼滤波的 Q 和 R 到底怎么调？"

### 6.3 追问按钮

每次 AI 回复后，气泡底部会出现 **3 个追问按钮**。这些问题是 AI 根据当前对话内容智能生成的，点击即可深入探讨。

### 6.4 消息撤回

发送消息后 **5 秒内**，你的消息气泡旁会出现一个半透明的 **「撤回」** 按钮：

- 点击「撤回」→ 消息取消发送，AI 停止回复，你的原文自动恢复到输入框
- 5 秒后按钮会自动淡出消失

这个功能在你打错字或发送了错误的指令时非常有用。

### 6.5 知识卡片联动

当 AI 的回答涉及某个知识点时，系统会自动在右侧侧边栏关联显示对应的知识卡片，包含核心概念、公式和常见误区。

### 6.6 双模式：Classic 与 Clar-ball (全新主动 AI)

在 AI 助教面板的顶部，你可以看到一个圆球形的模式切换开关，允许你在两种不同的 AI 辅导体验之间切换：

- **Classic 模式（被动响应）**：传统的 AI 聊天模式。Clar 会安静地待在右侧，只有在你主动提问或下发指令时，她才会给出解答。此时圆球表现为**绿色慢呼吸状态**。
- **Clar-ball 模式（智能感知与主动伴学）**：极简的智能体浮标小精灵模式。在此模式下，右侧聊天面板默认折叠隐藏，最大限度释放仿真屏幕空间。小精灵表现为发光的悬浮呼吸球，全过程伴随你的学习：
  
  **1. 呼吸状态灯指示**：
  - 🟢 **绿色慢呼吸**：Classic 模式下或 Clar-ball 模式空闲状态。
  - 🔵 **蓝色呼吸**：正在监听你的屏幕操作，等待你的鼠标悬停。
  - 🟣 **紫色高频呼吸**：正在对当前元素进行数据捕获和物理推演。
  - 🟢✨ **高亮强绿呼吸**：正在后台自动执行前端参数调整或仿真计算。

  **2. 屏幕感知交互 (1.5s 悬停)**：
  将鼠标悬停在输入参数框、Plotly 时频域图表、左下角导航菜单，或 ECharts 知识脑图节点上 **1.5 秒**。小精灵会自动进入高频紫光呼吸，并在你的鼠标当前位置浮现出一个高质感的毛玻璃 `.sensing-tooltip` 提示气泡。

  **3. 单击小球提问与默认窄版面板**：
  当小精灵感应完毕并跟随在你的鼠标旁时，**单击小球**，小球会自动向后台发送高情境感知的 Prompt，自动分析并讲解当前悬停元素的理论内涵、数学公式与常见误区。此时，AI 对话面板会自动滑出展开。为了防止遮挡你的信号图形，**面板默认以窄版模式（360px）展示**。你可以随时**双击面板空白处**将其切换为宽版（716px）。

  **4. 双击小球归位与模式切换**：
  - **小球归位**：当小精灵悬浮在屏幕中部感应时，**双击小球**会立即中止感知，并以平滑的抛物线动画飞回右下角初始锚定点（用时 0.8s ）。
  - **模式切换**：当小精灵处于右下角初始锚定点时，**双击小球**可以自由在 `Classic` 与 `Clar-ball` 模式之间进行双向切换。

  **5. 主动式物理卡片诊断**：
  如果在仿真实验中发生了异常物理状态，如：
  - 采样率设置过低，导致 Shannon-Nyquist 混叠
  - 滤波器截止频率与输入信号频宽不匹配
  - 卡尔曼滤波 Q/R 参数配比极端导致轨迹发散
  - 雷达追踪虚警或漏警率异常超标
  - 在重要知识节点无操作停留过久（学习停滞）
  
  小球会立即跳转为高频紫光呼吸，并在左侧弹出主动警告卡片（Proactive Action Card）。点击卡片上的**“查看诊断建议”**，AI 侧边栏会自动展开并直接呈递针对该物理问题的具体数学剖析与优化路线。

## 7. 进阶用法：让 AI 帮你操作

Clar 最强大的地方在于——你可以用自然语言直接让 AI 操作整个系统。以下是一些实用场景：

### 场景 1：一句话完成信号实验

```
你：生成一个 300Hz 的方波信号，信噪比 15dB
```
→ AI 自动：跳转信号工坊 → 填入参数 → 生成信号 → 显示波形

### 场景 2：一句话运行滤波

```
你：帮我做滑动平均滤波，窗口 30
```
→ AI 自动：跳转工具箱 → 设置窗口=30 → 运行滤波 → 显示对比图

### 场景 3：模糊指令也能执行

```
你：生成个信号看看
```
→ AI 自动用经典默认值生成正弦+白噪声信号

```
你：跑个低通滤波
```
→ AI 自动用默认截止频率 500Hz 运行 RC 低通滤波

### 场景 4：学习引导

```
你：我想了解维纳滤波的原理
```
→ AI：详细讲解维纳滤波 + 自动跳转到工具箱 + 运行维纳滤波仿真

### 场景 5：结合实验数据提问

```
你：（先生成一个窄带信号）
```

## 8. 常见问题 FAQ

### Q: 启动报错 "请在 .env 文件中设置有效的 DEEPSEEK_API_KEY"

**A**: 打开项目根目录下的 `.env` 文件，确保已填入有效的 DeepSeek API Key：
```
DEEPSEEK_API_KEY=sk-你的实际密钥
```

### Q: 页面打不开

**A**: 请确认：
1. 终端中有 `Uvicorn running on http://0.0.0.0:8001` 的输出
2. 浏览器访问的是 `http://localhost:8001`（注意端口是 **8001** 不是 8000）

### Q: AI 回复很慢或没有回复

**A**: 可能原因：
- DeepSeek API 调用延迟（首次连接可能需要几秒）
- 网络连接问题（检查你的网络是否畅通）
- API 额度用尽（检查 DeepSeek 账户余额）

### Q: 图表显示不出来

**A**: 请确保：
- 使用 Chrome 或 Edge 浏览器
- 网络可以访问 CDN 资源（Plotly.js, ECharts, KaTeX 通过 CDN 加载）
- 如果是离线环境，可能需要将 CDN 资源下载到本地

### Q: 工具箱显示"请先生成信号"

**A**: 工具箱中的滤波功能需要有信号数据作为输入。请先：
1. 切换到「信号工坊」页面
2. 生成一个信号
3. 再切换到「工具箱」运行滤波

或者直接告诉 AI：*"生成一个正弦信号然后做滑动平均滤波"*

### Q: 公式显示为乱码

**A**: 系统使用 KaTeX 渲染 LaTeX 数学公式，需要从 CDN 加载字体文件。请确保网络可以访问 `cdnjs.cloudflare.com`。

### Q: 如何同时让别人访问？

**A**: 使用 ngrok 进行内网穿透：
```bash
ngrok http 8001
```
将生成的 HTTPS 链接分享给对方即可。系统已做好 ngrok 兼容性适配（CORS、API 路径自动识别）。
